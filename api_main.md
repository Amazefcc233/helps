# APICenter
!> 本文档仍在施工中。

## 介绍

本APICenter是一个具有api功能的机器人。  
值得一提的是，所有**获得许可**的开发人员均可使用此APICenter。  
fcc事实上仅仅提供并维护了这一个平台。
 - 当前状态：**已开启**
 - 服务时间：2019.03.17-至今
 - 服务范围：QQ内不限范围
 
## 功能

目前APICenter支持以下功能。
 - 可提供fccの机器人全局帮助/help
 - 「msc限定」自助添加白名单
 - 「msc限定」查看服务器人数数据&远程执行命令
 - 「搜图姬限定」自助添加白名单
