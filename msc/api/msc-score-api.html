msc-api-v2

    msc-测试结果录入api-v2
    旧版本的api已不再有效，仅作为测试和参考使用

    接口URL

    https://cp.amazefcc233.com/msc/api/test_v1-2/api-v2.php

    请求方式

    POST

    请求 Content-Type

    application/json

    请求自定义Header

    参数 示例值 必填 参数描述

    请求Body参数

    {
    "id": "f_c_c",
    "Token": "SOME-TOKEN",
    "Type": "1",
    "Timestamp": "0"
}
    id : 玩家id，大小写不敏感，但推荐键入正确的id，必填
    Token : 管理员Token，将会在录入前进行验证。若验证失败，则当前次请求无效，必填（示例请求的SOME-TOKEN可用，但不会录入数据库）
    Type : 添加类型，请填写固定值1，非必填
    Timestamp : 时间戳，请填写固定值0，非必填

    响应参数

    参数 参数描述
    id 录入的游戏id
    Type 录入类型（默认为1）
    Timestamp 时间戳（默认为0）
    Status 状态，通常为“Ready”或“Debug-Mode”
    Debug （仅Debug模式下）Debug模式，一般为True
    Notice （仅Debug模式下）Debug模式提醒

    成功响应示例

    {
    "id": "f_c_c",
    "Type": "1",
    "Timestamp": "0",
    "Status": "Debug-Mode",
    "Debug": "True",
    "Notice": "当前正在Debug模式中，数据库未进行操作"
}

    失败响应示例

    {
    "Error": "错误码（和http状态码相同）",
    "Reason": "错误原因"
}

    msc-api-v1

    msc-测试结果录入api-v1
    此版本的api已不再有效，仅作为测试和参考使用

    接口URL

    https://cp.amazefcc233.com/msc/api/test_v1-2/api-v1.php?id=player_id

    请求方式

    POST

    请求 Content-Type

    Array

    请求自定义Header

    参数 示例值 必填 参数描述

    请求Body参数

    参数 示例值 必填 参数描述
    Token SOME-TOKEN 必填 管理员Token，若Token输入错误则当前次请求无效
    Timestrap 0 必填 时间戳，请返回固定值0
    Type 1 必填 类型，请返回固定值1

    请求Query参数

    参数 示例值 必填 参数描述
    id player_id 必填 玩家id，大小写不敏感

    成功响应示例

    成功应返回状态码 204 ，并不返回任何内容

    失败响应示例

    失败时应返回状态码 403 ，并返回Error信息
